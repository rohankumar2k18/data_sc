# AnomaData Project

## Introduction
This project aims to predict machine breakdown by identifying anomalies in the data. The dataset contains 18,000+ rows collected over a few days.

## Installation
1. Clone the repository:
   
   git clone https://github.com/yourusername/anomadata-project.git
Navigate to the project directory:

Copy code
cd anomadata-project
Install the required packages:

Copy code
pip install -r requirements.txt
Running the Code
Ensure you have the dataset in the project directory or provide the correct path in the script.
Run the main script:

Copy code
python main.py
